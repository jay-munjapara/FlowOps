# FlowOps

FlowOps is a Java full-stack workflow platform built with Spring Boot and Angular to automate approval workflows, containerize services with Docker, process real-time events with Kafka, and manage transactional data in PostgreSQL.

## Tech Stack

- Java
- Spring Boot
- Angular
- Docker
- Kafka
- PostgreSQL

## Features

- Workflow approval management
- REST API backend
- Angular frontend
- Dockerized services
- PostgreSQL persistence
- Event-driven workflow updates
